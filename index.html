• `$ git add -A` - Staged all of my perviously unstaged files/lines
  • `$ git commit -m ""` - This will take whatever is currently staged... and create a commit with it.
  • `$ git log` - With this command you can see your history of commits and the messages attached to them.
  • `$ git status` - Showing what files are staged or not staged (green is staged and ready to be committed)
